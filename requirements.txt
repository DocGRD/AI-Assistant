# Milestone 11 (zero-cost local Vault QA / RAG)
# Local embeddings (fastembed downloads BAAI/bge-small-en-v1.5 once, then offline)
# + a numpy vector index. No vault content leaves the machine.
# Indexing runs on the always-on Linux box, not the laptop.
# GPU box: run `install/install.sh --gpu` (or see Docs/DEPLOYMENT.md §1) — it installs the
#          CUDA-12 stack (fastembed-gpu + onnxruntime-gpu==1.22.0 + nvidia-*-cu12) and you set
#          "embedding_device": "cuda". Pin the CUDA-12 build — the latest targets CUDA 13.
fastembed
numpy
